Question -1 : What is the different ways to select on element on in the DoM?

There are several ways to select elements in the DOM. We can use 1 getElementById , 2 getElementsByClassName , 3 getElementsByTagName, 4 querySelectorAll 5 getElementsByName

Question -2 : What is the different between InnerHTML, InnerText and textContent?

InnerHTML give back the HTML of element, InnerText give back the text of an element and textContent gets all text inside the element.

Question -3 : What is event delegation in the dom?

ans: It is way that javascript allowed a parent element to handle events for its child element

Question -4 : What is event bubbling in the DOM?

Ans: Event bubbling is a process in the DOM where an event starts from the target element and propagates up through its ancestors (parent elements) until it reaches the document object.
